# Mokhtar Dashboard (Clean)

## Setup
1) npm install
2) Copy .env.local.example -> .env.local and fill Supabase URL + ANON KEY
3) In Supabase SQL editor run supabase/schema.sql
4) npm run dev then open http://localhost:3000

## Currency
All USD.

## Optional
- OneSignal push: set NEXT_PUBLIC_ONESIGNAL_APP_ID + ONESIGNAL_REST_API_KEY
- WhatsApp Cloud API: set WHATSAPP_TOKEN + WHATSAPP_PHONE_NUMBER_ID
